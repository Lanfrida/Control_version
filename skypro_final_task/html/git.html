Git

  • Git init - инициализирует новый репозиторий Git в текущей папке.
  • Git clone "url" - клонирует удалённый репозиторий на локальный Git
  • Git Add - добавляет все изменённые файлы.
  • Git commit -m "сообщение" - фиксирует изменения с комментарием.
  • Git log - выводит историю коммитов.
  • Git status - показывает состояние рабочей директории и индекса.
  • Git branch - список локальных веток.
  • Git branch "имя" - создаёт новую ветку.
  • Git checkout "ветка" - переключается на ветку.
  • Git pull - загружает изменения и сливает с текущей веткой
  • Git push - отправляет изменения в удалённый репозиторий.
